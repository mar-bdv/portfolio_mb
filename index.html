Интернет-магазин украшений "Sparkle"

Sparkle Shop — это демонстрационный интернет-магазин, созданный с использованием современных веб-технологий. Проект сочетает адаптивный дизайн, динамическую генерацию контента и удобное взаимодействие с API для аутентификации пользователей и управления заказами, создавая плавный и интуитивно понятный опыт для каждого.

Мебельный интернет-магазин

Этот проект — интернет-магазин мебели и товаров для дома, созданный на основе стека технологий PERN (PostgreSQL, Express, React и Node.js). В нём реализованы удобные списки товаров, корзина покупок и система регистрации пользователей. Магазин сочетает интуитивный интерфейс и надёжную работу с данными, обеспечивая удобство и простоту использования.
